Zendesk to Markdown

Input Zendesk Comment HTML

Output Markdown for Macro

How to Use:

  1. In Google Chrome, in Zendesk, Right-click on comment reply box -> "Inspect"
  2. In Google Chrome, Right-click on div class="zd-comment" -> "Edit as HTML"
  3. Select all, Copy all
  4. Paste into left box
  5. Click button
  6. Copy output into Zendesk's Macro box

Currently supports:

  • Bold
  • Italics
  • Lists: Ordered and Unordered
  • Headers
  • Quotes
  • Links
  • Div Removal
  • P Removal
  • Span Removal
  • Keeps &nbsp; (double spaces)!

NOT supported:

  • Nested quotes/lists (Zendesk doesn't allow anyways)
